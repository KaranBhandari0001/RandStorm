# ⚡ RandStorm

RandStorm is a powerful Browser Extension designed to generate random text, long strings, and structured paragraph data instantly.

It is built for developers, testers, and cybersecurity enthusiasts who frequently need dynamic input data during testing and development.

---

## 🚀 Why RandStorm?

During testing and development, especially while working with forms, APIs, and security tools, there is often a need to input different types of data:

* Short strings
* Long random data
* Continuous strings (no spaces)
* Paragraph-style input

Manually creating such data is time-consuming and inefficient.

👉 RandStorm was built to solve this exact problem.

---

## 🛡️ Purpose

This tool is also a small part of a larger cybersecurity recon toolkit I am building.

RandStorm acts as a utility module to generate diverse input data required during:

---

## ✨ Features

* Random Text Generator
* Ultra Long String Generator (up to 50,000 characters)
* Smart Paragraph Generation
* No Space Mode (continuous string generation)
* Custom Length Control
* Support for:

  * Uppercase (A-Z)
  * Lowercase (a-z)
  * Numbers (0-9)
  * Symbols
* Local Storage Support (auto-save + restore)
* One-click Copy
* Clear Output Option
* Fast & Lightweight

---

## 🖥️ Installation

1. Download or clone this repository

2. Open your browser and go to:

   ```
   chrome://extensions/
   ```

   (or equivalent extensions page in your browser)

3. Enable **Developer Mode**

4. Click **Load Unpacked**

5. Select the project folder
